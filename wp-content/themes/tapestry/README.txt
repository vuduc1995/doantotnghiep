Tapestry Child Theme
http://www.studiopress.com/themes/tapestry

INSTALL
1. Upload the Tapestry child theme folder via FTP to your wp-content/themes/ directory. The Genesis parent theme needs to be in the wp-content/themes/ directory as well.
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Tapestry theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

SOCIAL MEDIA ICONS
The icons that are used in the header right widget are being displayed with the Social Profiles Widget plugin, and can be downloaded from http://www.studiopress.com/plugins/social-profiles-widget .
