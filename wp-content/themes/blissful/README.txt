Blissful Child Theme
http://www.studiopress.com/themes/blissful

INSTALL
1. Upload the Blissful child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Blissful theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGETS
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar or Sidebar/Content Layout option.
Home Top - This is the top section of the homepage.
Home Bottom - This is the bottom section of the homepage.
Footer #1 - This is the top left column of the footer section.
Footer #2 - This is the top right column of the footer section.
Footer #3 - This is the bottom wide column of the footer section.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the blissful child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the blissful child theme demo site.

mini-thumbnail - 75px by 75px
small-thumbnail - 110px by 110px
