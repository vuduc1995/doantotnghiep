Crystal Child Theme
http://www.studiopress.com/themes/crystal

INSTALL
1. Upload the Crystal child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Crystal theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGETS
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Home Top Left - This is the top left section of the homepage.
Home Top Right - This is the top right section of the homepage.
Home Middle - This is the middles section on the homepage.
Home Bottom #1 - This is the first column of the bottom section of the homepage.
Home Bottom #2 - This is the second column of the bottom section of the homepage.
Home Bottom #3 - This is the third column of the bottom section of the homepage.

SLIDESHOW
The slideshow that is underneath the header is using the WP-Cycle plugin which can be downloaded at http://wordpress.org/extend/plugins/wp-cycle. Upload the wp-cycle folder via FTP to your wp-content/plugins/ directory and then activate it. You will also need to configure the plugin by going to the Plugins > WP-Cycle page in your dashboard.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Crystal child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Crystal child theme demo site.

Home Bottom - 65px by 65px
Home Thumbnail - 175px by 125px
Portfolio Thumbnail - 210px by 130px
Portfolio Full - 600px by 300px
