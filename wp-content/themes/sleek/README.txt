Sleek Child Theme
http://www.studiopress.com/themes/sleek

INSTALL
1. Upload the Sleek child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Sleek theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGETS
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Featured Left - This is the featured left column of the homepage.
Featured Right - This is the featured right column of the homepage.
Home Bottom #1 - This is the first column of the footer section.
Home Bottom #2 - This is the second column of the footer section.
Home Bottom #3 - This is the third column of the footer section.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Sleek child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Sleek child theme demo site.

Home Top Left - 600px by 314px
Home Top Right - 75px by 75px
Home Bottom - 280px by 150px

THEME COLORS
The Sleek child theme comes with 2 color styles - style.css and dark.css. To select the stylesheet which will get loaded, simply go to the Genesis > Theme Settings page in your WordPress dashboard and look for the Sleek Color Style option box.

PSD FILES
The Sleek child theme comes with PSD files for images used in the theme. You can download them at http://www.studiopress.com/support/showthread.php?t=33413.
