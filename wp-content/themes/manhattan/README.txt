Manhattan Child Theme
http://www.studiopress.com/themes/manhattan

INSTALL
1. Upload the Manhattan child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Manhattan theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

SLIDESHOW
The slideshow that is underneath the header is using the WP-Cycle plugin which can be downloaded at http://wordpress.org/extend/plugins/wp-cycle. Upload the wp-cycle folder via FTP to your wp-content/plugins/ directory and then activate it. You will also need to configure the plugin by going to the Plugins > WP-Cycle page in your dashboard.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Manhattan child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file.
