<a href="http://www.studiopress.com/themes/stretch">http://www.studiopress.com/themes/stretch</a>

<strong>INSTALL</strong>
1. Upload the Stretch child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Stretch theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

<strong>BACKGROUND</strong>
The default background image used in this theme can be found at <a href="http://subtlepatterns.com/low-contrast-linen/">http://subtlepatterns.com/low-contrast-linen/</a>.

To set the background image for your post or page, simply upload a photo through the WordPress image gallery and set it as the Featured Image.
