<strong>SOCIAL EYES CHILD THEME</strong>
<a href="http://www.studiopress.com/themes/socialeyes">http://www.studiopress.com/themes/socialeyes</a>

<strong>INSTALL</strong>
1. Upload the Social Eyes child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Social Eyes theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

<strong>WIDGETS</strong>
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Homepage - This is the content section of the homepage.
After Post Ad - This is the section after a post for an ad.
Footer #1 - This is the first column of the footer section.
Footer #2 - This is the second column of the footer section.
Footer #3 - This is the third column of the footer section.

<strong>FEATURED IMAGES</strong>
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Social Eyes child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Social Eyes child theme demo site.

Homepage Featured - 600px by 300px
Homepage Thumbnail - 110px by 110px

<strong>THEME COLORS</strong>
To utilize one of the color styles for the Social Eyes child theme, simply go to the Genesis > Theme Settings page in your WordPress dashboard and look for the Socialize Color Style option box. To customize any of the color styles, you will find the color-specific CSS at the very bottom of the style.css file.

<strong>SOCIAL MEDIA ICONS</strong>
The icons that are used in the header section of the Social Eyes demo site can be added using the Social Profiles widget found at <a href="http://wordpress.org/extend/plugins/social-profiles-widget">http://wordpress.org/extend/plugins/social-profiles-widget</a>.
