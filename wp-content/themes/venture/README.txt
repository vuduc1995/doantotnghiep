Venture Child Theme
http://www.studiopress.com/themes/venture

INSTALL
1. Upload the Venture child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Venture theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGETS
Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.
Header Tabs - This is the featured tabbed section of the homepage.
Home Left - This is the left section of the homepage.
Home Right - This is the right section of the homepage.
Footer Left - This is the left section of the footer.
Footer Right - This is the right section of the footer.

FEATURED IMAGES
By default WordPress will create a default thumbnail image for each image you upload and the size can be specified in your dashboard under Settings > Media. In addition, the Venture child theme creates the following thumbnail images you'll see below, which are defined (and can be modified) in the functions.php file. These are the recommended thumbnail sizes that are used on the Venture child theme demo site.

Mini - 90px by 90px
Featured - 400px by 300px
